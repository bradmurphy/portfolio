The first major campaign I was tasked with upon arriving at Hook. This was part of a dynamic campaign that featured over 14 different banners with different sizes all using dynamic content. Using Doubleclick, we were able to create a dynamic feed spreadsheet that populates each iteration of the banner with seemingly random data and copy targeted towards specific markets.

All in all, this was a great campaign to work on -- after many interations and pixel perfect adjustments, we managed to deploy all banners shortly before Google redesigned their logo.
